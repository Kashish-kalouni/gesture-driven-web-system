# Gesture-Control Website

A gesture-driven interactive web system that allows users to control website navigation and interactions using hand gestures detected via webcam.

## Tech Stack
- **React.js + TypeScript**
- **MediaPipe Hands**
- **react-webcam**
- **Tailwind CSS**

## Features
- Navigate pages using hand gestures
- Touchless and interactive interface
- Add and view images controlled by gestures
- Simple, hygienic, and accessible UI

## How to Run
  1. Clone the repo:
  ```bash
  git clone <repo-link>
  cd gesture-control-website
  
  2. Install dependencies:
  npm install
  
  3. Start the development server:
  npm start
  
  4. Open in browser: http://localhost:3000.
